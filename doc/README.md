## クイックリファレンス

### ロール別アクセス可能画面

| 画面 | 管理者 | 営業担当者 | 閲覧者 |
|---|:---:|:---:|:---:|
| ダッシュボード | ✓ | ✓ | ✓ |
| 顧客一覧 | ✓ | ✓ | ✓ |
| 顧客登録・編集 | ✓ | ✓ | — |
| 商談一覧 | ✓ | ✓ | ✓ |
| 商談登録・編集 | ✓ | ✓ | — |
| 設定（自分の表示名） | ✓ | ✓ | ✓ |
| ユーザー管理（`/admin/users`） | ✓ | — | — |

### 主要 URL

```
/              ダッシュボード
/customer      顧客一覧
/deal          商談一覧
/settings      ユーザー設定
/admin/users   ユーザー管理（管理者専用）
/setup         初回表示名設定（未設定ユーザーのみ）
/pending       権限付与待ち（未割り当てユーザーのみ）
```
